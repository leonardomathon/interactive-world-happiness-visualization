2IMV20 - Visualization Assignment

This visualization project is built with Javascript using several packages, among others D3.js, THREE.js, TailwindCSS and Webpack. It was built for the course 2IMV20 - Visualization


Webpack is used as a bundler, that bundles the assets (such as js and css files). The output of Webpack can be found in the /dist folder. If you are editing files, make sure you are working within the /src directory, otherwise Webpack will not see the changes you have made.


The project also has live (hot) reloading, to start the live reloading server use the command npm run start. Make sure all the dependencies are correctly installed! Now open a webbrowser and go to the url localhost:8080.
